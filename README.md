# AI Code Explainer

## Project Description
AI Code Explainer is a web-based application that allows users to input code snippets and get simple, human-readable explanations using AI.

The system supports both:
- Online AI Model (API-based)
- Local AI Model

It helps developers quickly understand code logic and improve productivity.

---

## Objective
Build a tool that:
- Accepts code input (Java, JavaScript, etc.)
- Generates plain-English explanation
- Allows multiple chat sessions
- Provides clean UI with chat history

---

## Tech Stack

###  Backend
- Java
- Spring Boot
- REST API
- WebClient (for API calls)

### Frontend
- HTML
- CSS
- Bootstrap 5
- JavaScript

##  Features

✅ AI-based code explanation
